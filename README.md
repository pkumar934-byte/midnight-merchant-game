# Midnight Merchant
A mind-blowing Python prototype blending cozy Shop Simulation with intense Survival Horror. Built using Pygame-CE.

## How to play
1. Navigate to the project folder.
2. Run the game using the packaged environment: `.\venv\Scripts\python.exe main.py`

## Rules
- **Day Phase:** Move the player using `W, A, S, D` or `Arrow Keys`. Wait at the bottom counter for customers to spawn. Walk up to them and press `SPACE` to sell an item from your inventory to gain Gold. The Day Phase lasts 10 seconds.
- **Night Phase:** The screen will go dark. Ghosts will spawn and relentlessly track your position. Use your unsold inventory as weapons by pressing `SPACE` to unleash a magical blast to banish nearby ghosts. Be careful, if a Ghost touches you, it's Game Over!
